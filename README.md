## Kratki opis funkcionalnosti

Aplikacija omogućava korisnicima personalizirano praćenje vijesti putem RSS feedova kroz sljedeće značajke:

- **Upravljanje feedovima:** Korisnici mogu dodati RSS feedove iz različitih izvora vijesti (portali, blogovi, tech news).
- **Tematske grupe:** Omogućuje organizaciju feedova u personalizirane kategorije (npr. "Tech", "HR vijesti", "Globalne vijesti", "Sport"....) Postoji nekoliko pre-generiranih feedova za korisnike ( OVO SE UGLAVNOM KORISTI ).
- **Osnovne informacije ->** korisnici imaju pristup:
  - Naslovu vijesti
  - Izvoru (RSS feed - link na originalni article)
  - Opisu i sadržaju članka
  - Datumu objave
  - Poveznici na originalni članak
  - Slici/thumbnail-u
  - Kategoriji kojoj pripada
- **AI Daily Spotlight:** Aplikacija dnevno generira:
  - Sažetak najvažnijih vijesti
  - AI-generirani pregled svih ( nekoliko ) -> top vijesti is svake kategorije ( news feed-a od korisnika )
  - Personalizirane preporuke članaka
- **Cross-device sync:** Sve feedovi sync-ani na svim uređajima!
